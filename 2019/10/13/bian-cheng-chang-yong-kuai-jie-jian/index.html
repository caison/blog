Intellij idea快捷键
编辑类快捷键
Ctrl + Shift + Alt + U 生成UML图(专业版才有的功能)
Ctrl + Alt + L 代码自动缩进
Ctrl + Alt + T 对选中的代码块进行快速包装
Ctrl + Alt + V 自动补全变量名称、返回值
Ctrl + Alt + O 优化imports（去除不必要的）
Alt + Shift + ↑/↓ 选中代码上移/下移一行

查看快捷键
Ctrl + Alt + ← 光标回到上一次的位置
Ctrl + Alt + → 光标回到下一次的位置
Ctrl + Alt + S 打开设置
Alt + Q 快速查看当前方法的声明
Ctrl + Q 查看方法说明

Ctrl + 左键变量 跳到变量声明的地方
Ctrl + Shift + F10 运行文件
Ctrl + Shift + F 全局搜索
Ctrl + E 打开最近打开的文件
Ctrl + Shift + Num 添加数字书签(数字必须是键盘横排上的数字)
Ctrl + num 快速跳转到指定的数字书签(数字必须是键盘横排上的数字)
Ctrl + R 替换
Ctrl + Y 删除行
Ctrl + D 复制当前行到下一行
Ctrl + Shift + R 全局替换
Ctrl + Shift + Alt + S 打开项目属性配置
Ctrl + H + 点击名词 查看详细
Ctrl + 空格 弹出建议列表
Ctrl + Shift + V 打开粘贴的历史列表
Ctrl + Shift + N 快速打开文件
Ctrl + N 快速搜索类
Ctrl + F9 重新编译
Ctrl + E 最近浏览历史
Ctrl + Q 查看方法声明

Ctrl + W 选你所想
Alt + Insert 快速添加get和set方法
Alt + 1 打开项目视图
Alt + 6 TODO
Shift + F6 重新命名
Shift + Shift 万能搜索
Shift + Enter 任意位置换行
psvm + Tab 创建main方法
for + i 快速写for循环
sout 快速写System.out.println()

查看函数列表
在Project框中最右边有一个齿轮状的设置按钮，增加“show members”即可
